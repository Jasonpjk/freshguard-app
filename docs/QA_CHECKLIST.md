# FreshGuard — QA 체크리스트

> 최종 업데이트: 2026-05-23

## 1. 대시보드 (Dashboard)

- [ ] 소비기한 현황 카드 6개 모두 표시됨
- [ ] 입고/개봉 현황 카드 4개 표시됨 (오늘 입고, 미개봉, 개봉 후 기한 임박, 오늘 개봉)
- [ ] 우선처리 테이블에서 "사용 완료" 클릭 → 해당 품목 상태가 정상으로 변경됨
- [ ] 우선처리 테이블에서 "폐기" 클릭 → DisposalDialog 열림
- [ ] DisposalDialog에서 폐기 등록 → 폐기 기록 화면에 반영됨
- [ ] "전체 보기" 버튼 클릭 → 품목 관리 화면으로 이동

## 2. 품목 관리 (ItemsManagement)

- [ ] 품목 목록 로드됨
- [ ] 검색창 입력 → 실시간 필터링 동작
- [ ] 상태 탭 필터 (전체/만료/임박/주의/정상) 동작
- [ ] "품목 추가" 버튼 → 추가 모달 열림
- [ ] 추가 모달 필수 항목 누락 시 저장 불가
- [ ] 품목 추가 → 목록에 즉시 반영, localStorage에 저장됨
- [ ] 편집 아이콘 클릭 → 수정 모달 열림
- [ ] 수정 저장 → 목록에 즉시 반영됨
- [ ] 삭제 버튼 → 확인 후 삭제됨

## 3. 입고/개봉 관리 (StockManagement)

- [ ] 화면 로드 시 요약 통계 카드 4개 표시됨
- [ ] 탭 필터 (전체/미개봉/개봉/사용완료/폐기) 동작
- [ ] 검색창으로 품목명/카테고리/위치 검색 가능
- [ ] "입고 등록" 버튼 → 모달 열림
- [ ] 입고 모달 필수항목(품목명, 소비기한, 수량) 누락 시 오류 토스트
- [ ] 입고 등록 완료 → 테이블에 미개봉 상태로 추가됨
- [ ] 미개봉 품목 → "개봉" 버튼 표시됨
- [ ] "개봉" 버튼 클릭 → 개봉 처리 모달 열림
- [ ] 개봉 처리 완료 → 해당 품목 개봉일 업데이트, 소비기한 재계산됨
- [ ] 개봉 품목 → "사용완료" 버튼 클릭 → 사용완료 상태로 변경됨
- [ ] "폐기" 버튼 → 폐기 처리 및 폐기 기록 추가됨
- [ ] QR 아이콘 표시 (qrLabelEnabled=true 품목)

## 4. 소비기한 캘린더 (ExpiryCalendar)

- [ ] 현재 월 달력 표시됨
- [ ] 소비기한이 있는 날짜에 배지 표시됨
- [ ] 날짜 클릭 → 해당일 품목 목록 표시됨
- [ ] 이전/다음 달 이동 동작

## 5. 폐기 기록 (DisposalRecords)

- [ ] 폐기 기록 목록 로드됨
- [ ] 승인/대기 상태 배지 표시됨
- [ ] 월별 총 손실금액 표시됨
- [ ] 대기 상태 기록 → 승인/거부 버튼 동작

## 6. 보관 위치 관리 (StorageLocations)

- [ ] 위치 카드 표시됨 (냉장고, 냉동고, 건식창고, 바냉장고)
- [ ] 카드 클릭 → 해당 위치 품목 목록 펼쳐짐
- [ ] "위치 추가" → 추가 모달 동작
- [ ] 편집/삭제 동작

- [ ] 모든 필수 항목 완료 시 "점검 완료" 성공 토스트
- [ ] 점검 이력 패널 표시됨

- [ ] 역할/상태 필터 동작
